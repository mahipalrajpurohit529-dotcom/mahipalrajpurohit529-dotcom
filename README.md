## 🐍 `contribution_snake.py`

<div align="center">

![github-snake](https://raw.githubusercontent.com/mahipalrajpurohit529-dotcom/mahipalrajpurohit529-dotcom/output/github-snake-dark.svg)

</div>

> **To activate the snake:** Create `.github/workflows/snake.yml` in this repo with the following:
>
> ```yaml
> name: Generate Snake
> on:
>   schedule:
>     - cron: "0 0 * * *"
>   workflow_dispatch:
> jobs:
>   build:
>     runs-on: ubuntu-latest
>     steps:
>       - name: Generate Snake
>         uses: Platane/snk@v3
>         with:
>           github_user_name: mahipalrajpurohit529-dotcom
>           outputs: |
>             dist/github-snake.svg
>             dist/github-snake-dark.svg?palette=github-dark
>       - name: Push to output branch
>         uses: crazy-max/ghaction-github-pages@v3.1.0
>         with:
>           target_branch: output
>           build_dir: dist
>         env:
>           GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
> ```
> Then go to **Actions tab → run workflow manually once.**
